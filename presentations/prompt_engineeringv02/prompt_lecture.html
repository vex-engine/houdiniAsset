> Grok → Midjourney → Nano Banana 실전 파이프라인

할머니도 할수있는
프롬프트!

// 15분 안에 끝내는 AI 레퍼런스 워크플로우 — 2026.04

가벼운 인사 후 바로 시작. "오늘 15분 동안 보여드릴 건 AI로 그림 그리는 법이 아니라, 우리가 매일 하는 콘티/레퍼런스 작업을 10배 빠르게 만드는 파이프라인입니다. 끝나고 두 가지를 가져가시면 됩니다 — 시니어분들은 '파이프라인 관점', 주니어분들은 '내일 바로 써먹을 한 줄'."

슬라이드 띄우자마자 영상 바로 재생. 영상 끝나고 "이거 만드는 데 ○○시간 걸렸습니다. 옛날 같으면 며칠 걸릴 작업이에요." 시간 수치는 실제 데이터로 채워주세요.

> 핵심 메시지

프롬프트는
명령어가 아니라 콘티다

AI한테 말하는 법 = 프로그래밍

AI한테 말하는 법 = 후배 아티스트한테 콘티 주기

  • 우리는 이미 콘티 짜는 법을 안다
  • 라이팅, 카메라, 재질, 분위기 — 매일 쓰는 언어
  • 이걸 그대로 옮기면 됨
"프롬프트 엔지니어링이라는 단어 때문에 어렵게 느껴지지만, 본질은 콘티입니다. 우리가 외주 컨셉 작가한테 일 맡길 때 뭐라고 하시나요? '낮 시간대, 와이드샷, 폐허 분위기, 동양적 디테일' — 이게 이미 프롬프트입니다."

> 1단계: Grok

왜 컨셉은 그록인가

파이프라인 관점
  • 검열이 약함 → 다크 판타지/폐허/전투씬 막힘 없음
  • 대화형 반복 → 한 세션에서 30개 변형 누적
  • Speed 모드 → 탐색 비용 사실상 0
  • 포지션: "발산 단계 전용"
지금 당장 한 줄
기획서 한글 텍스트 그대로 그록에 던지고:
"이걸 영어 이미지 프롬프트 5개 변형으로 만들어줘. 분위기는 each 다르게."
→ 영어 작성 부담 0, 5개 비교 가능
"주니어분들 영어 프롬프트 쓰는 게 부담이시죠? 그록은 한글로 던져도 알아서 영어 프롬프트를 만들어줍니다. 그리고 한 번에 5개씩 변형을 요청하는 게 핵심이에요."

그록으로 30분 만에 컨셉 30장

환경 5개 → 라이팅 5개 → 카메라 5개  |  한 번에 하나만 바꾼다

영상 재생 전: "여기서 보실 건 — 한 번에 모든 걸 바꾸지 않고, 한 가지 변수만 바꿔가면서 수렴하는 과정입니다."

미드저니에서 스타일을 가져온다

MJ: 시네마틱 톤 압도적  +  Grok: 대화형 수정  →  두 장점만 합친다

"이게 진짜 핵심 트릭입니다. 미드저니는 톤이 압도적인데 수정이 답답해요. 그록은 수정은 잘 되는데 톤이 평범해요. 그래서 미드저니에서 톤만 한 장 뽑고, 그걸 그록에 레퍼런스로 넣어버립니다."

> 3단계: Nano Banana

나노바나나로 정밀 수정

파이프라인 관점
  • 강점: 정확한 수정 지시가 먹힘
  • 레퍼런스 이미지 14장까지 입력 가능
  • "여기 안개", "이 부분 어둡게" 같은 부분 수정에 강함
  • 포지션: "최종 마무리 편집기"
지금 당장 한 줄
"이 이미지 그대로 두고, 왼쪽 기둥 옆에 횃불 추가해줘. 빛은 따뜻한 주황색."
→ 부분만 정확히 바뀜
"그록에서 컨셉 잡고, 미드저니에서 톤 잡았으면, 마지막은 나노바나나에서 디테일 수정입니다. 나노바나나의 킬러 기능은 부분 수정이에요."

실제 프로젝트 적용

Grok 컨셉 → 3D 블록아웃 → UE4 라이팅 → 인게임

> 주의

이렇게 쓰면 망합니다

❌ 한 번에 다 바꾸기

bad.txt
"카메라 부감으로, 라이팅 밤으로,
 분위기 음산하게, 안개 추가,
 횃불 추가해줘"

→ 원래 이미지와 완전 다른 거 나옴

✅ 한 번에 하나만

good.txt
라운드 1: "카메라만 부감으로"
라운드 2: (결과 보고) "라이팅 밤으로"
라운드 3: (결과 보고) "안개 추가"

→ 원하는 방향으로 정확히 수렴

원칙: 한 번에 하나만 바꾼다

"이게 가장 흔한 실수입니다. 욕심 부리지 마세요. AI는 한 번에 하나만 바꿀 때 가장 정확합니다."

> 정리

프롬프트는
후배 아티스트한테 주는 콘티다

  • 단계마다 다른 도구 — Grok(발산) → MJ(톤) → Nano Banana(조립)
  • 한 번에 하나만 바꾼다 — 점진적 수렴
  • 한글로 던져도 된다 — 그록한테 영어 변환 시키기

내일 할 일: 한 줄 카드 중 1개만 골라서 써보기

"마지막으로 — 이건 우리를 대체하는 도구가 아닙니다. 반복적 탐색 작업을 AI에 맡기고, 우리는 판단과 디테일에 집중하는 거예요. 여러분의 20년, 10년 경력은 'AI가 만든 걸 보고 뭐가 옳은지 판단하는 눈'에 들어 있습니다."
