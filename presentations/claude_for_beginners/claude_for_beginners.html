01

클로드가 대체 뭔데?

제미나이 쓰던 사람, 손 들어봐요 ✋

분위기 풀기: "혹시 제미나이 써본 사람?" 손 들게 유도

> 01 · 클로드 vs 제미나이

솔직히 말해줄게요 👀

🔵 제미나이

  • 구글 계정만 있으면 OK
  • 구글 검색이랑 연동 굿
  • 유튜브 요약도 돼요
  • 답이 좀 딱딱한 느낌
  • 가끔 너무 짧게 답해요

✨ 클로드

  • 대화가 훨씬 자연스러워요
  • 긴 글 쓰기에 압도적
  • 창의적 아이디어 탁월
  • 시각화·분석 최강 신기능
  • 공감 능력이 진짜 사람 같아
비교표를 보여주며 "느낌이 달라요. 클로드는 친구 같고, 제미나이는 선생님 같아요"

> 01 · 핵심 요약

제미나이 = 정보 검색
클로드 = 생각 파트너

숙제 도움받을 땐 제미나이
아이디어, 글쓰기, 분석은 클로드

"둘 다 좋아요. 근데 오늘은 클로드 집중!"

02

이렇게만 쓰면 돼요

프롬프트? 그냥 말 잘 거는 법 🗣️

프롬프트를 어렵게 생각하지 말라고, "그냥 톡 하듯이 하면 돼요"

> 02 · 프롬프트 기초

프롬프트 = AI한테 보내는 카톡

❌ 이렇게 하면 망해요

"알바 그만두고 싶어"

→ 클로드가 뭘 원하는지 몰라요 😅

✅ 이렇게 하면 대박

"나 27살 편의점 알바 중인데,
월 50만원 더 버는 부업 아이디어
5개만 알려줘. 스마트폰만 있으면 되는 걸로"

→ 맞춤 답변 바로 출력! 🎯

"구체적일수록 좋은 답이 와요. 카톡도 '배고파'보다 '치킨 먹고 싶어, 교촌 가자'가 낫잖아요"

> 02 · 황금 프롬프트 공식

이 3가지만 넣으면 끝 🏆

👤
나는 누구
"27살 편의점 알바생"
🎯
뭘 원해
"부업 아이디어 5개"
📌
조건이 뭔데
"스마트폰만, 하루 1시간"

// 나 + 원하는 것 + 조건 = 완벽한 프롬프트

"이 공식 외우세요. 진짜 이게 전부예요."

> 02 · 실전 예시

알바하면서 바로 써먹는 프롬프트 📱

클로드에 이렇게 입력 
나는 27살 편의점 알바생이야.
야간 알바 마치고 집에서 스마트폰으로
하루 1~2시간만 써서 할 수 있는 부업을
현실적으로 월 50만원 목표로 알려줘.
초보도 바로 시작할 수 있는 것 위주로,
각 부업마다 시작 방법도 같이 알려줘.

→ 클로드가 맞춤 부업 리스트 + 시작 가이드를 한 번에 써줘요 🎉

직접 클로드 열어서 시연해도 좋음

> 02 · 꿀팁

클로드는 이어서 대화가 돼요 💬

일반 검색

검색 → 결과 확인
또 검색 → 또 확인
맥락 없이 처음부터 반복
😫 피곤해

클로드 대화

답변 받음
"이거 좀 더 자세히" → 바로 이어서
"이건 빼고 다시 써줘" → OK
"더 쉽게 설명해줘" → 당연하죠
😎 맥락 다 기억함!

// 채팅하듯이 계속 다듬으면 돼요

"한 번에 완벽할 필요 없어요. 대화처럼 주고받으면서 다듬으면 됩니다"

> 03 · 시각화 활용 아이디어

이런 것들 만들어서 팔 수 있어요 💸

  • 인스타 인포그래픽 제작 대행
  • 유튜브 썸네일 아이디어 제안
  • 사업계획서 차트·도표 제작
  • 포트폴리오 시각 자료 제작
  • 독서 요약 마인드맵 판매
  • 스터디 자료 시각화 제작
  • 블로그 콘텐츠 도표 제작
  • 크몽·탈잉 강의 자료 제작
💡 현실 수익

인스타 인포그래픽 1개 = 3~5만원 · 하루 2개 제작 가능 = 월 150만원+

"실제로 크몽에서 인포그래픽 제작 서비스 많아요. 클로드 쓰면 30분이면 만들어요"

> 03 · 💰 부자 되는 프롬프트 #1

인스타 인포그래픽 제작 자동화

복붙해서 바로 써요 
주제: "20대 직장인이 월 100만원 모으는 법"
 
이 주제로 인스타그램 인포그래픽에 쓸
핵심 포인트 5가지를 뽑아줘.
각 포인트마다:
- 임팩트 있는 숫자나 통계 포함
- 한 줄 설명 (15자 이내)
- 관련 이모지 1개
그리고 전체 레이아웃 구성도 제안해줘.

→ 나온 내용을 캔바(Canva)에 붙여넣으면 → 완성! 🎨

캔바는 무료 디자인 툴. 클로드 + 캔바 조합이 최강

> 03 · 💰 부자 되는 프롬프트 #2

재테크 마인드맵 시각화

복붙해서 바로 써요 
월급 200만원 받는 27살이
5년 안에 1억 모으는 재테크 전략을
마인드맵 형태로 시각화해줘.
 
중심 주제: "5년 안에 1억 만들기"
1단계 가지: 수입 늘리기 / 지출 줄이기 / 투자하기
각 가지마다 구체적인 실천 방법 3가지씩
텍스트로 구조화해서 보여줘.
실천 가능한 현실적인 방법으로.

→ 마인드맵 앱(Miro, XMind)에 옮기면 → 판매 가능한 콘텐츠 완성!

마인드맵은 네이버 블로그, 인스타, 크몽에 팔 수 있음

> 03 · 💰 부자 되는 프롬프트 #3

수익 분석 차트 자동 생성

복붙해서 바로 써요 
내가 시작할 수 있는 SNS 부업 5가지를
비교 분석해줘.
 
비교 항목:
- 시작 비용 (0원~10만원 구간으로)
- 월 예상 수익 (현실적으로)
- 준비 기간
- 난이도 (★ 1~5개)
 
결과를 표 형태로 정리하고,
초보에게 가장 추천하는 것 1개 강조해줘.

→ 나온 표를 그대로 블로그·유튜브 정보 콘텐츠로 활용!

"이 표 하나가 블로그 조회수 폭발하는 콘텐츠가 됩니다"
