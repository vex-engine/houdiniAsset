02

컨텍스트(맥락)

// 프롬프트의 핵심

> 같은 단어, 다른 그림

맥락이
결과를 결정한다

  • "abandoned castle" → 서양 중세? 일본 성? 판타지?
  • 낮? 밤? 새벽? 멀리서? 내부? 문 앞?
  • AI는 모호함을 "평균"으로 처리한다
  • 구체적 맥락 = 원하는 결과의 유일한 방법

6가지 맥락 레이어

레이어설명예시

03

핵심 키워드 마스터

> 공식

Subject → Description → Style

formula.txt
[공간 유형], [세계관/시대],[핵심 오브젝트 2~3개],[라이팅 조건],[카메라 앵글 + 렌즈],[재질감 키워드],[분위기 형용사],[렌더 스타일]

키워드 사전 ① 환경

  • 자연 — volcanic wasteland, frozen tundra, crystal cavern, floating islands
  • 인공 — crumbling Gothic cathedral, abandoned industrial complex, underground bunker
  • 복합 — castle ruins reclaimed by forest, sunken city half-submerged in swamp

// 두 개를 섞으면 AI가 독창적인 해석을 내놓는다

키워드 사전 ② 라이팅

  • 시간대 — golden hour, blue hour, harsh midday, overcast diffuse, twilight
  • 광원 — volumetric god rays, bioluminescent glow, campfire warm point light
  • 분위기 — dramatic chiaroscuro, neon-lit cyberpunk, candlelit warm interior

// UE4에서 라이트 세팅할 때 생각하는 것들을 영어로 옮기면 된다

키워드 사전 ③ 카메라 & 재질

카메라/앵글

wide-angle establishing (14mm)
bird's eye / top-down
low-angle looking up
deep / shallow depth of field

재질/표면

weathered stone with moss
rusted corroded metal
wet cobblestone reflecting light
polished obsidian, frosted glass

> 주의

2026년에
효과 없는 키워드

더 이상 안 먹히는 "품질 태그"

masterpiece, best quality, highly detailed, 8k, ultra-realistic, trending on ArtStation

대안: 구체적 디테일 묘사가 범용 태그보다 훨씬 효과적

네거티브 프롬프트

negative.txt
blurry, low resolution, watermark, text overlay,cartoon style, anime, flat lighting,oversaturated colors, lens flare,people in foreground, UI elements

04

반드시 알아야 할 주의사항

> 주의사항 ①

저작권

  • 안전 레퍼런스/무드보드 용도로 활용
  • 안전 AI 생성 이미지 기반으로 인간이 수정/재창작
  • 리스크 AI 생성물을 최종 게임 에셋으로 직접 사용
  • 금지 특정 아티스트 이름으로 스타일 복제

> 주의사항 ②

기술적 정확성의 한계

  • 구조적 지지 — 물리적으로 불가능한 건축물
  • 비율과 스케일 — 문 크기 불일치
  • 재질 물리 — 유리가 돌처럼, 금속이 천처럼
  • 양식 혼합 — 고딕+아르데코+일본식 무분별 혼합

// AI가 제안하고, 우리가 검증하고 실현한다

> 주의사항 ③

일관성 유지의 어려움

  • 레퍼런스 이미지 활용 — 이전 결과를 다음 입력으로
  • 스타일 가이드 프롬프트 — 색상/재질/라이팅 고정 문구
  • 시드(Seed) 고정 — 시드 번호 고정하여 변수 최소화
  • 프롬프트 템플릿 — 팀 공유 라이브러리 구축

프롬프트 초안 생성

prompt.txt
다음 게임 배경 설정을이미지 생성 AI용 영어 프롬프트로 변환해줘.스타일은 언리얼 엔진 시네마틱 렌더 느낌으로.[설정] 화산 지대 한가운데 위치한 고대 대장간.용암이 천연 화로 역할을 한다...

> Step 2

대량 변형 생성

  • 1라운드 — 구도 — wide / medium / close-up
  • 2라운드 — 라이팅 — dawn / midday / dusk / night
  • 3라운드 — 분위기 — ominous / serene / chaotic

// 한 라운드에 하나의 변수만 바꿔가며 비교하라

> Step 3 ~ 4

선별 → UE4 연결

Step 3: 보드 구성

PureRef에 카테고리별 분류. 팀 리뷰 → 방향성 합의.

Step 4: UE4 작업

블록아웃 → 라이팅 → 머티리얼 방향성.

실전 프롬프트 3종

  • 다크 판타지 던전 — vast underground throne room, obsidian pillars, volumetric dust
  • 자연 + 폐허 — ancient aqueduct overgrown with giant roots, misty ravine, golden hour
  • SF 산업 시설 — abandoned space mining facility, harsh directional sunlight, hard sci-fi

// 환경, 라이팅, 분위기 부분만 바꿔도 수십 개 변형 가능

활용 제안

  • 프롬프트 라이브러리 — 프로젝트별 검증된 템플릿 공유
  • 레퍼런스 생성 규칙 — 레퍼런스 용도 한정, AI 태그 표시

> 정리

핵심 3가지

  • AI는 레퍼런스 도구다 — 최종 에셋이 아니라 방향성 탐색의 가속기
  • 맥락이 품질을 결정한다 — 공간, 라이팅, 카메라, 재질, 분위기를 구체적으로
  • 반복이 실력이다 — 한 번에 하나씩 바꾸며 빠르게 수렴
