# 단축키 레퍼런스

최종 업데이트: 2026-04-24

## 🖥️ 프레젠테이션 모드 (익스포트된 HTML 파일)

### 페이지 이동 (한 번에 한 페이지)
| 키 | 동작 |
|---|---|
| `→` / `↓` / `Arrow Right` / `Arrow Down` | 다음 페이지 |
| `←` / `↑` / `Arrow Left` / `Arrow Up` | 이전 페이지 |
| `Page Down` | 다음 페이지 |
| `Page Up` | 이전 페이지 |
| `Home` | 첫 페이지 |
| `End` | 마지막 페이지 |

### 스텝(블록) 단위 진행
| 키 | 동작 |
|---|---|
| `Space` | 블록 하나씩 진행 → 마지막 블록 후 다음 페이지 |

### 마우스 휠
| 입력 | 동작 |
|---|---|
| 휠 아래로 | 블록 하나씩 나타남 → 마지막 블록 후 다음 페이지 (진입 시 블록 전부 숨김) |
| 휠 위로 | 블록 하나씩 사라짐 → 첫 블록 전이면 이전 페이지 (진입 시 블록 전부 표시) |
| 첫 페이지에서 휠 위로 | 동작 안 함 (맨 처음) |
| 첫 페이지 | 항상 전체 블록 표시 상태 |

### 터치 (모바일)
| 동작 | 결과 |
|---|---|
| 왼쪽으로 스와이프 | 다음 페이지 |
| 오른쪽으로 스와이프 | 이전 페이지 |

### 기타
| 키 | 동작 |
|---|---|
| `F` | 전체화면 토글 |
| `N` | 발표자 노트 표시 토글 |
| `E` | 에디터 모드 진입/종료 (원본에서만) |

## ✏️ 에디터 모드

### 페이지 이동
| 키 | 동작 |
|---|---|
| `Page Down` | 다음 슬라이드 |
| `Page Up` | 이전 슬라이드 |
| 마우스 휠 ↓ (캔버스 위) | 다음 슬라이드 |
| 마우스 휠 ↑ (캔버스 위) | 이전 슬라이드 |

### 파일 / 저장
| 키 | 동작 |
|---|---|
| `Ctrl + S` | 원본 파일 저장 |
| `Ctrl + Shift + S` | 프레젠테이션용 파일 익스포트 |

### 편집 히스토리
| 키 | 동작 |
|---|---|
| `Ctrl + Z` | 실행 취소 (텍스트 편집 중엔 브라우저 기본 동작) |
| `Ctrl + Shift + Z` | 다시 실행 |
| `Ctrl + Y` | 다시 실행 |

### 요소 조작 (선택된 블록에 대해)
| 키 | 동작 |
|---|---|
| `Ctrl + D` | 선택 요소 복제 |
| `↑` `↓` `←` `→` | 1px 이동 |
| `Shift + 방향키` | 10px 이동 (NUDGE_LG) |
| `Delete` / `Backspace` | 선택 요소 삭제 (텍스트 편집 중엔 브라우저 기본 동작) |

### 표시 옵션
| 키 | 동작 |
|---|---|
| `G` | 그리드 표시 토글 |
| `E` | 에디터 모드 진입/종료 |

### 입력 다이얼로그 (프롬프트 창)
| 키 | 동작 |
|---|---|
| `Enter` | 확인 |
| `Escape` | 취소 |

---

## 💡 동작 요약

**프레젠테이션 모드**
- 키보드 방향키/PageUp/PageDown → **페이지 단위** 이동
- Space 및 마우스 휠 → **블록 단위** 진행 (빌드 애니메이션 체험용)

**에디터 모드**
- 모든 페이지 이동 수단(키/휠) → **페이지 단위** 즉시 이동
- 블록별 애니메이션은 익스포트 후에만 의미 있음

---

## 🔒 안전장치

에디터 모드의 휠 네비게이션은 다음 상황에서는 동작하지 않음:
- `contenteditable` 텍스트 편집 중 (글자 스크롤이 필요할 수 있음)
- 블록 드래그/리사이즈 중 (`.ed-dragging`, `.ed-resizing`)
- `<input>` 또는 `<textarea>` 포커스 중
- 사이드바 영역 (슬라이드 프레임 밖)
